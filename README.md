# MetaHorse

## 專案簡介

以賽馬遊戲為主題的NFT購買平台，使用Vue框架練習前端開發

### Demo

https://kelvinlin9.github.io/metahorse/#/

## 主要功能

### 前台
- 商品陳列、分類陳列、商品搜尋、單一商品介紹
- 購物車（新增項目、編輯數量、刪除項目、套用優惠券）
- 優惠活動（抽獎）
- 訂單查詢
- 結帳、結帳後小遊戲

### 後台
- 管理者登入 / 登出、驗證登入狀態
- 後台管理
  - 訂單管理
  - 訂單統計圖表
  - 產品管理
  - 優惠券管理
